# Learn Hub-Online Learning Platform (MERN Stack)
Learn Hub is a full-stack web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). 
The platform serves as a centralized learning hub where users can browse, enroll in, and manage online courses. 
Designed to simulate a real-world e-learning system, this project demonstrates modern web development practices, user authentication, dynamic content, and RESTful API integration.

This project helps in understanding full-stack development, API handling, authentication, and real-time data interaction using MongoDB.
It is a great learning resource for anyone who wants to build and deploy a full-stack application from scratch.
